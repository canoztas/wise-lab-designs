Discovering Security Risks of Android Apps Using Metadata

This project focuses on leveraging metadata from Android markets—including app definitions, scores, and user feedback—to identify security vulnerabilities. Unlike traditional distribution models, Android's centralized approach enables systematic analysis of application metadata to uncover potential risks.

Sponsored by Tubitak
