WISE

Wireless Networks and Intelligent Secure Systems Lab conducts research on wireless networks, IoT and security. Our research is at the intersection of machine learning, nature inspired algorithms and their applications to the hard problems in networking and security. The main aim of the group is to develop intelligent secure systems. We are particularly interested in malware analysis, anomaly and intrusion detection, and trust management.

Projects

Secure Protocols in IoT

In this project, IoT protocols will be aimed to be analyzed from the security perspective and be strengthened. It is a cooperative work with Dr. Manel Guerrero-Zapata and Ruken Zilan from Universitat Politècnica de Catalunya (UPC) - Barcelona Tech.

UpDroid

Update attacks are one of the biggest threats against mobile devices today. Existing mobile security solutions are generally ineffective in detecting such attacks. The main objective of this project is to analyze updated attacks, to create a dataset of such attacks, to detect such attacks by using static and dynamic analysis based detection techniques.

Co-Evolution of Malwares and Anti-Malware Systems

As long as anti-malware software develops, malware writers also develop their malicious code by using various evasion strategies such as obfuscation and encryption. This is the lifecycle of malicious and anti-malware software. In this study, the use of evolutionary computation techniques are investigated, both for developing new variants of mobile malwares and for developing better security solutions against them automatically.

CoIoT

Coevolutionary computation techniques for developing intrusion detection models for IoT networks utilizing IPv6 routing protocol in low power lossy networks (RPL).

Detecting Intrusions in IIoT

Industrial IoT will replace many wired/wireless industrial components and will be required to be highly reliable and secure. Intrusion detection systems are key components in securing IIoT.

DeSRe

Discovering Security Risks of Android Apps using Metadata. Leveraging app definitions, scores, and user feedback from Android markets to identify security vulnerabilities.
